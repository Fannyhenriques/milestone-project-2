Memory Match

Game Rules

Test your memory by flipping two cards at a time to find matching pairs.

If the cards match, they stay face up. If not, they will flip back over.

You start the game with 100 points. Each time you miss a pair, you lose 5 points.

The game ends when all pairs are found. Try to keep your score as high as possible!

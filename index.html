Memory Match

This memory game was made for a school project at Code Institute during Spring 2026.

It is built using HTML, CSS, and JavaScript, and demonstrates basic interactive features such as card matching, scoring, and end-of-game messages.

Developed by Fanny Henriques.

Game Rules

Test your memory by flipping two cards at a time to find matching pairs.

If the cards match, they turn green and stay face up. If not, they will flip back over.

You start the game with 100 points. Each time you miss a pair, you lose 5 points.

Pay attention to your score: the color changes as your points drop (from green to orange, then red) to help you keep track.

The game ends in one of two ways: either you find all the pairs and win, or your points reach 0 and the game ends.

Try to keep your score as high as possible!
